¿Por qué nace ProWork Analytics?

ProWork Analytics nace como un proyecto académico orientado a dar solución a una problemática frecuente en entornos productivos: la falta de control estructurado sobre la asistencia, las tareas y la productividad laboral.

Especialmente en la industria textil, muchos procesos se gestionan de forma manual o con herramientas dispersas, dificultando el análisis real del rendimiento y la toma de decisiones. Este sistema busca centralizar la información y convertir los datos en indicadores claros.

Propósito del sistema

  • Gestión de operarios, líneas y tareas
  • Control de asistencia por turnos
  • Registro de tiempos reales de ejecución
  • Análisis mediante KPIs industriales
  • Reportes históricos y visualización gráfica

Alcance

ProWork Analytics está diseñado para la industria manufacturera ligera, con especial enfoque en la industria textil. Integra módulos administrativos, operativos y analíticos bajo una arquitectura escalable.

Base tecnológica

  • Java + Spring Boot
  • HTML, CSS, JavaScript, Thymeleaf
  • Chart.js para visualización de datos
  • PostgreSQL
  • Arquitectura MVC
